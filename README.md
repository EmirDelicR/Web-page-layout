### How to use

In utils/constant.js file are constants:

- **_NORMAL_PAGES_**
- **_SMALL_PAGES_**
- **_SOCIAL_PAGES_**

1. Here set the names of your pages

2. Add the page names to translations

3. Add components and import them in specific view

That is all
